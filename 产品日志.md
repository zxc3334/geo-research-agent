# GeoResearch Agent — 产品开发日志

> 从研究项目到可面试的 Agent 产品的开发记录。

## 产品定位

**GeoResearch — 面向 GIS/遥感领域的 AI 深度研究平台**

- 垂直 SaaS 产品（Web UI，面向研究者）
- Deep Research API 服务（面向开发者和企业）
- 差异化：evidence-aware 信息分级 + 领域专用验证工具 + 红蓝对抗质量保证

## 技术栈

| 层       | 技术                                  | 说明                         |
| -------- | ------------------------------------- | ---------------------------- |
| Frontend | Next.js 14 + TypeScript + Tailwind    | Web 界面                     |
| Backend  | FastAPI + Pydantic v2                 | HTTP API                     |
| 异步任务 | Celery + Redis                        | 长任务队列                   |
| 数据库   | PostgreSQL + SQLAlchemy 2.0 + Alembic | 持久化                       |
| 缓存     | Redis                                 | 会话缓存、速率限制、任务状态 |
| 部署     | Docker Compose                        | 一键启动                     |
| 实时推送 | SSE (Server-Sent Events)              | 研究进度                     |
| 认证     | JWT + API Key                         | 用户 + 开发者                |

这个回答展示了你**清楚知道差什么、按什么顺序做、每步多少时间**，而不是"我还需要部署"这种模糊说法。
